How ours service works

Capturing and Sending Garbage Image

The user takes a photo of garbage using a mobile app, which sends the image and location data to a central server for further processing and action.

Processing the Request on the Server

The server verifies the image and location, identifies the appropriate area, and sends a cleaning request to the responsible corporator for further action and delegation.

Assigning Cleaning Workers

The corporator receives the request, assigns cleaning workers to the location, and ensures the area is cleaned. Workers update the system once the cleaning task is completed.
